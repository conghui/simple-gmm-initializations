: algo description file
: ----------------------
:
: Needs file extension '.algo'.
:
: Indicate comments by leading ':'.
:
: Define the EM algorithm by 
:	first:em:<steps range spec>:<seed range spec>
: where a <... range spec> is given by
: 	minimumValue-maximumValue.stepSize
: or
:       value1, value2, ..., valueLast
:
: e.g.,

first:em:50:1-3.1
